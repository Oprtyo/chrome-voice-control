# Voice Control / Голосовое управление Chrome

Управляйте браузером Chrome голосовыми командами на русском языке.

## Как это работает

1. Python-сервер с **Vosk** постоянно слушает микрофон (локально, офлайн)
2. При фразе **«Окей гугл»** открывается окно распознавания — говорите команду
3. Команда распознаётся через **Google Speech API** и выполняется в браузере
4. При фразе **«Открой браузер»** — сразу запускается Chrome (даже если закрыт)

## Установка

### 1. Установите зависимости

**Windows:**
```
install.bat
```

**Linux / macOS:**
```bash
chmod +x install.sh start.sh
./install.sh
```

Скрипт установит Python-библиотеки и скачает модель Vosk (~45 МБ).

### 2. Установите Chrome-расширение

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика** (правый верхний угол)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите эту папку (`chrome-voice-control`)

### 3. Запустите сервер

**Консольный режим:**
- Windows: `start.bat`
- Linux / macOS: `./start.sh`

**Режим трея (свёрнутый, с правами администратора):**
- Windows: `start_tray.vbs` (полностью скрытый, запросит UAC) или `start_tray.bat`
- Иконка микрофона появится в трее — правый клик для управления
- Запуск от администратора необходим для управления WireGuard VPN

**Автозапуск при старте Windows:**
1. `Win+R` → `shell:startup` → Enter
2. Скопируйте ярлык на `start_tray.vbs` в открывшуюся папку

В popup расширения появится зелёный индикатор «Сервер подключён».

## Использование

### Wake-фразы

| Фраза | Действие |
|-------|----------|
| **«Окей гугл»** | Активирует распознавание команды через Google Speech API |
| **«Открой браузер»** | Запускает Chrome (работает даже без открытого браузера) |

### Голосовые команды (после «Окей гугл»)

| Команда | Действие |
|---------|----------|
| **назад** | Назад в истории |
| **вперёд** | Вперёд в истории |
| **обновить** | Перезагрузить страницу |
| **домой** | Новая вкладка |
| **вверх / вниз** | Прокрутка |
| **влево / вправо** | Горизонтальная прокрутка |
| **в начало / в конец** | К верху / низу страницы |
| **новая вкладка** | Открыть новую вкладку |
| **закрыть вкладку** | Закрыть текущую |
| **следующая / предыдущая вкладка** | Переключение |
| **найди [запрос]** | Поиск в Google |
| **открой [сайт]** | Открыть сайт |
| **увеличить / уменьшить** | Зум страницы |
| **нажми** | Клик по центру экрана |
| **на весь экран** | Полноэкранный режим (нажатие F — YouTube и др.) |
| **включи VPN** | Включить WireGuard VPN |
| **выключи VPN** | Выключить WireGuard VPN |
| **выбери ссылку** | Показать номера на ссылках, затем назвать номер для клика |
| **отмена** | Убрать номера ссылок |

Все команды и фразы можно изменить в `config.json`.

## Настройка

Все настройки хранятся в файле `config.json` в корне проекта:

```json
{
  "wake_phrase": "окей гугл",
  "launch_phrase": "открой браузер",
  "server": {
    "port": 9876
  },
  "chrome": {
    "path": "C:\\Program Files\\Google\\Chrome\\Application\\chrome.exe",
    "profile": "Profile 1",
    "flags": ["--silent-debugger-extension-api"]
  },
  "wireguard": {
    "path": "C:\\Program Files\\WireGuard\\wireguard.exe",
    "tunnel": "1MyVPN-Laptop"
  },
  "commands": { ... }
}
```

**Что можно настроить:**
- `wake_phrase` — фраза активации голосовых команд
- `launch_phrase` — фраза для запуска Chrome
- `chrome.path` — путь к Chrome
- `chrome.profile` — профиль Chrome
- `chrome.flags` — флаги запуска Chrome
- `wireguard.path` — путь к WireGuard
- `wireguard.tunnel` — имя туннеля WireGuard
- `commands` — все голосовые команды (можно добавлять/менять/удалять)

Выбор микрофона:
```
python server\voice_server.py --device 1
```

## Требования

- **Python 3.8+**
- **Google Chrome 116+**
- **Микрофон**
- Интернет для первой установки (скачивание модели) и для распознавания команд (Google Speech API)
- **Права администратора** для управления WireGuard VPN

## Структура проекта

```
chrome-voice-control/
├── config.json              # Все настройки (команды, пути, фразы)
├── manifest.json            # Chrome extension manifest (v3)
├── background/              # Service worker
├── content/                 # Content script (действия на странице)
├── offscreen/               # WebSocket клиент (постоянное соединение)
├── recognizer/              # Окно распознавания (Google Speech API)
├── popup/                   # UI расширения
├── icons/                   # Иконки
├── server/
│   ├── voice_server.py      # Python сервер (Vosk + WebSocket)
│   ├── tray_server.py       # Трей-версия сервера
│   ├── requirements.txt     # Python зависимости
│   └── model/               # Модель Vosk (после установки)
├── install.bat / .sh        # Скрипт установки
├── start.bat / .sh          # Скрипт запуска (консоль)
├── start_tray.bat / .vbs    # Скрипт запуска (трей, от администратора)
└── README.md
```
